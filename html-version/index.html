Robots.txt – это специальный файл, расположенный в корневом каталоге сайта. Веб-мастер указывает в нем, какие страницы и данные не следует индексировать. Файл содержит директивы, описывающие доступ к разделам сайта (так называемый стандарт исключений для роботов). Например, с его помощью можно создать отдельные настройки доступа для поисковых роботов, предназначенных для мобильных устройств и обычных компьютеров. Справка Яндекса и Гугла
